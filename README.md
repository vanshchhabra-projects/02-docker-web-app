# Dockerized Web Application

A simple website deployed inside a Docker container using Nginx.

## Run Locally
docker build -t docker-web-app .
docker run -d -p 8081:80 docker-web-app

## Tech Stack
- Docker
- Nginx
- HTML
- CSS
